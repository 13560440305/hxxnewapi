# 快速开始

> 生成 API 密钥并进行首次调用。

请先将星游开放平台部署并启动（参见项目 README），并在管理后台创建 API Key。下文中的 `<API_KEY>` 替换为你的密钥；接口地址已使用 **https://www.hxxopen.com**，若自建部署请改为你的网关地址。

## 直接调用 API

```py
import requests
import json

response = requests.post(
    url="https://www.hxxopen.com/v1/chat/completions",
    headers={
        "Authorization": "Bearer <API_KEY>",
        "Content-Type": "application/json",
    },
    data=json.dumps({
        "model": "gpt-4o-mini",  # 替换为你在后台配置的模型 id
        "messages": [
            {
                "role": "user",
                "content": "你好，请用一句话介绍你自己。"
            }
        ]
    })
)
print(response.json())
```

```bash
curl 'https://www.hxxopen.com/v1/chat/completions' \
  -H 'Authorization: Bearer <API_KEY>' \
  -H 'Content-Type: application/json' \
  -d '{
    "model": "gpt-4o-mini",
    "messages": [
      { "role": "user", "content": "你好，请用一句话介绍你自己。" }
    ]
  }'
```

支持流式调用时，在请求体中增加 `"stream": true` 即可。

## 使用 OpenAI SDK

将 `base_url` 指向你的网关地址，`api_key` 使用在管理后台创建的密钥。

```py
from openai import OpenAI

client = OpenAI(
    base_url="https://www.hxxopen.com/v1",
    api_key="<API_KEY>",
)

completion = client.chat.completions.create(
    model="gpt-4o-mini",
    messages=[
        {"role": "user", "content": "你好，请用一句话介绍你自己。"}
    ],
    temperature=0.7,
    max_tokens=1024,
)
print(completion.choices[0].message.content)
```

```js
import OpenAI from 'openai';

const openai = new OpenAI({
  baseURL: 'https://www.hxxopen.com/v1',
  apiKey: '<API_KEY>',
});

const completion = await openai.chat.completions.create({
  model: 'gpt-4o-mini',
  messages: [
    { role: 'user', content: '你好，请用一句话介绍你自己。' }
  ],
});
console.log(completion.choices[0].message);
```
