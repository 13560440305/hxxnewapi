<span id="get-models"></span>
## 获取所有模型元数据

获取所有模型元数据。

**权限**：需要管理员权限（Admin）。

**请求方式**：`GET`  
**路径**：`/api/models/`

### Authorization

使用 Bearer Token 认证（管理端 Access Token）。

- **格式**：`Authorization: Bearer <token>`
- **位置**：`header`

### Response

- **200**：成功时返回模型元数据列表（具体格式以实际接口为准）。

### 示例

<!-- tabs:start -->
#### **cURL**
```bash
curl -X GET "https://www.hxxopen.com/api/models/" \
  -H "Authorization: Bearer <API_KEY>"
```

#### **JavaScript**
```js
fetch("https://www.hxxopen.com/api/models/", {
  method: "GET",
  headers: {
    "Authorization": "Bearer <API_KEY>"
  }
})
```

#### **Python**
```py
import requests

response = requests.get(
    "https://www.hxxopen.com/api/models/",
    headers={"Authorization": "Bearer <API_KEY>"},
)
print(response.json())
```

**权限**：需要管理员权限（Admin）。

**请求方式**：`GET`  
**路径**：`/api/models/{id}`

### Authorization

- **格式**：`Authorization: Bearer <token>`
- **位置**：`header`

### Path Parameters

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| id | integer | 是 | 模型 ID |

### Response

- **200**：成功时返回该模型的元数据（具体格式以实际接口为准）。

### 示例

```bash
  -H "Authorization: Bearer <API_KEY>"
```

#### **JavaScript**
```js
fetch("https://www.hxxopen.com/api/models/0", {

})
```

#### **Python**

```py
import requests

response = requests.get(
    "https://www.hxxopen.com/api/models/0",
```

<span id="search-models"></span>
## 搜索模型

按关键词搜索模型。

**权限**：需要管理员权限（Admin）。

**请求方式**：`GET`  
**路径**：`/api/models/search`

### Authorization

使用 Bearer Token 认证（管理端 Access Token）。

- **格式**：`Authorization: Bearer <token>`
- **位置**：`header`

### Query Parameters

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| keyword | string | 否 | 搜索关键词 |

### Response

- **200**：成功时返回匹配的模型列表（具体格式以实际接口为准）。

### 示例

<!-- tabs:start -->
#### **cURL**
```bash
curl -X GET "https://www.hxxopen.com/api/models/search?keyword=gpt" \
  -H "Authorization: Bearer <API_KEY>"
```

#### **JavaScript**
```js
fetch("https://www.hxxopen.com/api/models/search?keyword=gpt", {
  method: "GET",
  headers: {
    "Authorization": "Bearer <API_KEY>"
  }
})
```

#### **Python**
```py
import requests

response = requests.get(
    "https://www.hxxopen.com/api/models/search",
    params={"keyword": "gpt"},
    headers={"Authorization": "Bearer <API_KEY>"},
)
print(response.json())
```
