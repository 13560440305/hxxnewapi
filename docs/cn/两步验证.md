<span id="backup-codes"></span>
## 重新生成备用码

用于重新生成 2FA 备用码。

**权限**：需要登录（User 权限）。

**请求方式**：`POST`  
**路径**：`/api/user/2fa/backup_codes`

### Authorization

使用 Bearer Token 认证（用户 Access Token）。

- **格式**：`Authorization: Bearer <token>`
- **位置**：`header`

### Response

- **200**：成功时返回备用码重置结果（具体格式以实际接口为准）。

### 示例

<!-- tabs:start -->
#### **cURL**
```bash
curl -X POST "https://www.hxxopen.com/api/user/2fa/backup_codes" \
  -H "Authorization: Bearer <API_KEY>"
```

#### **JavaScript**
```js
fetch("https://www.hxxopen.com/api/user/2fa/backup_codes", {
  method: "POST",
  headers: {
    "Authorization": "Bearer <API_KEY>"
  }
})
```

#### **Python**
```py
import requests

response = requests.post(
    "https://www.hxxopen.com/api/user/2fa/backup_codes",
    headers={"Authorization": "Bearer <API_KEY>"},
)
print(response.text)
```

<span id="disable-2fa"></span>
## 禁用2FA

用于禁用当前用户的 2FA。

**权限**：需要登录（User 权限）。

**请求方式**：`POST`  
**路径**：`/api/user/2fa/disable`

### Authorization

使用 Bearer Token 认证（用户 Access Token）。

- **格式**：`Authorization: Bearer <token>`
- **位置**：`header`

### Request Body

`Content-Type: application/json`

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| code | string | 否 | 2FA 验证码 |

### Response

- **200**：成功时返回禁用结果（具体格式以实际接口为准）。

### 示例

<!-- tabs:start -->
#### **cURL**
```bash
curl -X POST "https://www.hxxopen.com/api/user/2fa/disable" \
  -H "Authorization: Bearer <API_KEY>" \
  -H "Content-Type: application/json" \
  -d '{"code":"123456"}'
```

#### **JavaScript**
```js
fetch("https://www.hxxopen.com/api/user/2fa/disable", {
  method: "POST",
  headers: {
    "Authorization": "Bearer <API_KEY>",
    "Content-Type": "application/json"
  },
  body: JSON.stringify({ code: "123456" })
})
```

#### **Python**
```py
import requests

response = requests.post(
    "https://www.hxxopen.com/api/user/2fa/disable",
    headers={
        "Authorization": "Bearer <API_KEY>",
        "Content-Type": "application/json",
    },
    json={"code": "123456"},
)
print(response.text)
```
