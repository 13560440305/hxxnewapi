# 实时语音

本页说明实时语音接口：通过 WebSocket 建立连接，用于实时对话。

**注意**：此为 WebSocket 端点，需使用 WebSocket 协议连接。

**连接方式**：WebSocket  
**路径**：`/v1/realtime`  
**连接 URL 示例**：`wss://www.hxxopen.com/v1/realtime?model=gpt-4o-realtime`

### Authorization

使用 Bearer Token 认证。可在连接时通过 URL 查询参数传递，或通过首条消息携带（视网关实现而定）。

- **格式**：`Authorization: Bearer <token>`（例如 `Authorization: Bearer sk-xxxxxx`）
- **常见方式**：在 WebSocket 请求头或连接 URL 中携带，如 `wss://www.hxxopen.com/v1/realtime?model=gpt-4o-realtime`，Token 需按网关要求传入（如 query 参数 `authorization` 或 header）

### Query Parameters

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| model | string | 否 | 要使用的实时语音模型 |

### Response

- **101 Switching Protocols**：成功升级为 WebSocket 连接，后续通过 WebSocket 收发消息进行实时对话。
- **400**：请求错误时返回 `application/json`：

```json
{
  "error": {
    "message": "string",
    "type": "string",
    "param": "string",
    "code": "string"
  }
}
```

### 示例

<!-- tabs:start -->
#### **JavaScript (WebSocket)**
```js
const token = "YOUR_API_KEY"
const model = "gpt-4o-realtime"
const url = `wss://www.hxxopen.com/v1/realtime?model=${encodeURIComponent(model)}`

const ws = new WebSocket(url, ["Bearer", token])

ws.onopen = () => console.log("已连接")
ws.onmessage = (e) => console.log("收到:", e.data)
ws.onerror = (e) => console.error("错误:", e)
ws.onclose = (e) => console.log("关闭:", e.code, e.reason)
```

#### **Python**
```py
import asyncio
import websockets

async def main():
    token = "YOUR_API_KEY"
    model = "gpt-4o-realtime"
    uri = f"wss://www.hxxopen.com/v1/realtime?model={model}"
    extra_headers = {"Authorization": f"Bearer {token}"}
    async with websockets.connect(uri, extra_headers=extra_headers) as ws:
        async for msg in ws:
            print("收到:", msg)

asyncio.run(main())
```
