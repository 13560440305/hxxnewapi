<span id="creem-webhook"></span>
## Creem Webhook

用于接收 Creem 的支付回调（Webhook）。

**鉴权**：无需鉴权（Webhook 回调）。

**请求方式**：`POST`  
**路径**：`/api/creem/webhook`

### Response

- **200**：成功时返回处理结果（具体格式以实际接口为准）。

### 示例

<!-- tabs:start -->
#### **cURL**
```bash
curl -X POST "https://www.hxxopen.com/api/creem/webhook"
```

#### **JavaScript**
```js
fetch("https://www.hxxopen.com/api/creem/webhook", {
  method: "POST"
})
```

#### **Python**
```py
import requests

response = requests.post("https://www.hxxopen.com/api/creem/webhook")
print(response.text)
```

<span id="user-creem-pay"></span>
## 发起 Creem 支付

发起 Creem 支付流程。

**权限**：需要登录（User 权限）。

**请求方式**：`POST`  
**路径**：`/api/user/creem/pay`

### Authorization

使用 Bearer Token 认证（用户 Access Token）。

- **格式**：`Authorization: Bearer <token>`
- **位置**：`header`

### Response

- **200**：成功时返回支付发起结果（具体格式以实际接口为准）。

### 示例

<!-- tabs:start -->
#### **cURL**
```bash
curl -X POST "https://www.hxxopen.com/api/user/creem/pay" \
  -H "Authorization: Bearer <API_KEY>"
```

#### **JavaScript**
```js
fetch("https://www.hxxopen.com/api/user/creem/pay", {
  method: "POST",
  headers: {
    "Authorization": "Bearer <API_KEY>"
  }
})
```

#### **Python**
```py
import requests

response = requests.post(
    "https://www.hxxopen.com/api/user/creem/pay",
    headers={"Authorization": "Bearer <API_KEY>"},
)
print(response.text)
```
