# 模型

本页说明模型列表接口：获取当前可用的模型列表。

根据请求头与参数自动识别返回格式：

- 请求头包含 `x-api-key` 和 `anthropic-version` 时，返回 Anthropic 格式
- 请求头包含 `x-goog-api-key` 或查询参数带 `key` 时，返回 Gemini 格式
- 其他情况返回 OpenAI 格式

**请求方式**：`GET`  
**路径**：`/v1/models`

### Authorization

使用 Bearer Token 认证。

- **格式**：`Authorization: Bearer <token>`（例如 `Authorization: Bearer sk-xxxxxx`）
- **位置**：`header`

### Query Parameters

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| key | string | 否 | Google API Key，用于请求 Gemini 格式时使用 |

### Header Parameters

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| x-api-key | string | 否 | Anthropic API Key，用于请求 Claude 格式 |
| anthropic-version | string | 否 | Anthropic API 版本 |
| x-goog-api-key | string | 否 | Google API Key，用于请求 Gemini 格式 |

### Response

- **200**：成功时返回 `application/json`，OpenAI 格式示例：

```json
{
  "object": "list",
  "data": [
    {
      "id": "gpt-4",
      "object": "model",
      "created": 0,
      "owned_by": "openai"
    }
  ]
}
```

- **401**：未授权时返回：

```json
{
  "error": {
    "message": "string",
    "type": "string",
    "param": "string",
    "code": "string"
  }
}
```

### 示例

<!-- tabs:start -->
#### **cURL**
```bash
curl -X GET "https://www.hxxopen.com/v1/models" \
  -H "Authorization: Bearer <API_KEY>"
```

#### **JavaScript**
```js
fetch("https://www.hxxopen.com/v1/models", {
  method: "GET",
  headers: {
    "Authorization": "Bearer <API_KEY>"
  }
})
```

#### **Python**
```py
import requests

response = requests.get(
    "https://www.hxxopen.com/v1/models",
    headers={"Authorization": "Bearer <API_KEY>"},
)
print(response.json())
```
