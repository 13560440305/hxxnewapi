## 文本转语音

将文本转换为音频。

**请求方式**：`POST`  
**路径**：`/v1/audio/speech`

### Authorization

使用 Bearer Token 认证。

- **格式**：`Authorization: Bearer <token>`（例如 `Authorization: Bearer sk-xxxxxx`）
- **位置**：`header`

### Request Body

`Content-Type: application/json`

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| model | string | 是 | 模型 ID（如 `tts-1`） |
| input | string | 是 | 要转换的文本，长度 ≤ 4096 |
| voice | string | 是 | 音色：`alloy` \| `echo` \| `fable` \| `onyx` \| `nova` \| `shimmer` |
| response_format | string | 否 | 默认 `mp3`。可选：`mp3` \| `opus` \| `aac` \| `flac` \| `wav` \| `pcm` |
| speed | number | 否 | 语速，默认 `1`，范围 0.25 ~ 4 |

### Response

- **200**：成功时返回音频流，`Content-Type` 为 `audio/mpeg`（或与 `response_format` 对应类型）。

### 示例

<!-- tabs:start -->
#### **cURL**
```bash
curl -X POST "https://www.hxxopen.com/v1/audio/speech" \
  -H "Authorization: Bearer <API_KEY>" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "tts-1",
    "input": "你好，这是一段测试文本。",
    "voice": "alloy"
  }'
```

#### **JavaScript**
```js
const body = JSON.stringify({
  "model": "tts-1",
  "input": "你好，这是一段测试文本。",
  "voice": "alloy"
})

fetch("https://www.hxxopen.com/v1/audio/speech", {
  method: "POST",
  headers: {
    "Content-Type": "application/json",
    "Authorization": "Bearer <API_KEY>"
  },
  body
})
```

#### **Python**
```py
import requests

response = requests.post(
    "https://www.hxxopen.com/v1/audio/speech",
    headers={
        "Authorization": "Bearer <API_KEY>",
        "Content-Type": "application/json",
    },
    json={
        "model": "tts-1",
        "input": "你好，这是一段测试文本。",
        "voice": "alloy",
    },
)
# response.content 为音频二进制数据
with open("output.mp3", "wb") as f:
    f.write(response.content)
```

## 音频转录

将音频转换为文本（语音识别，STT）。

**请求方式**：`POST`  
**路径**：`/v1/audio/transcriptions`

### Authorization

使用 Bearer Token 认证。

- **格式**：`Authorization: Bearer <token>`（例如 `Authorization: Bearer sk-xxxxxx`）
- **位置**：`header`

### Request Body

`Content-Type: multipart/form-data`

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| file | file | 是 | 音频文件（binary） |
| model | string | 是 | 模型 ID（如 `whisper-1`） |
| language | string | 否 | ISO-639-1 语言代码 |
| prompt | string | 否 | 提示文本 |
| response_format | string | 否 | 默认 `json`。可选：`json` \| `text` \| `srt` \| `verbose_json` \| `vtt` |
| temperature | number | 否 | 温度参数 |
| timestamp_granularities | array\<string\> | 否 | 时间戳粒度 |

### Response

- **200**：成功时返回 `application/json`，如 `{ "text": "string" }`。

### 示例

<!-- tabs:start -->
#### **cURL**
```bash
curl -X POST "https://www.hxxopen.com/v1/audio/transcriptions" \
  -H "Authorization: Bearer <API_KEY>" \
  -F file="@/path/to/audio.mp3" \
  -F model="whisper-1"
```

#### **JavaScript**
```js
const formData = new FormData()
formData.append("file", audioFile)  // File 对象
formData.append("model", "whisper-1")

fetch("https://www.hxxopen.com/v1/audio/transcriptions", {
  method: "POST",
  headers: {
    "Authorization": "Bearer <API_KEY>"
  },
  body: formData
})
```

#### **Python**
```py
import requests

with open("audio.mp3", "rb") as f:
    response = requests.post(
        "https://www.hxxopen.com/v1/audio/transcriptions",
        headers={"Authorization": "Bearer <API_KEY>"},
        files={"file": f},
        data={"model": "whisper-1"},
    )
print(response.json())
```
